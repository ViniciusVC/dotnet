
## Projeto Csharp_API.

# Criar projeto .net API

dotnet new webapi --name csh arp_api

# Rorando servidor 

dotnet watch run

Ou com atualização automatica.

dotnet watch run

# Ao iniciar o servidor 

Abre uma pagina em com Swagger 6.5 em:

http://localhost:5271/swagger/index.html
