# Instal dotnet core

## 1º Instalar o SDK do .NET:

sudo apt-get install -y dotnet-sdk-7.0

## 2º Crie uma pasta para o projeto:

mkdir projetoteste

## 3º Entre na pasta:

cd projetoteste

## 4º Iniciei um projeto de terminal:

dotnet new console

## 5ª Rodar a aplicação:

dotnet run
