# Criar projeto .net API
```
$ dotnet new webapi --name csharp_api
```

# Rorando servidor 
```
$ dotnet watch run
ou
$ dotnet run

```

# Ao iniciar o servidor 
```
Abra uma pagina em com Swagger em:
http://localhost:5271/swagger/index.html
```

# Controllers
```
Apage os  arquivos controllers criados por padrão.
```
